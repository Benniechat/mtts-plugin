A comprehensive Learning Management System for Mountain-Top Theological Seminary.

== Description ==

The MTTS LMS Plugin provides a complete suite of tools for managing the seminary's academic operations.

**Features:**
*   **Student Dashboard:** Course registration, fee payments, exam taking, result checking.
*   **Lecturer Dashboard:** Class management, attendance, assignment grading, CBT question bank.
*   **Admissions:** Online application workflow with automated processing.
*   **CBT System:** Built-in Computer Based Test engine with auto-grading, timer, and randomization.
*   **Payments:** Integration with Paystack and Flutterwave.
*   **Zoom Integration:** Schedule and join virtual classes.
*   **Alumni Network:** Directory, jobs, and events for graduates.
*   **Documents:** Auto-generate Admission Letters, ID Cards, Transcripts, and Certificates.

== Installation ==

1. Upload the `mtts-lms` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure settings in the 'MTTS LMS' -> 'Settings' menu.
4. Ensure Permalinks are set to 'Post name'.
